# Crypto

A Java-based Cryptography Toolkit that implements both Classical and Modern Encryption Algorithms with graphical user interfaces (GUI).

## Features

* AES Encryption and Decryption
* RSA Encryption and Decryption
* Caesar Cipher
* Affine Cipher
* Playfair Cipher
* Hill Cipher
* Vigenère Cipher
* One-Time Pad Cipher
* Rail Fence Cipher
* Row Transposition Cipher
* GUI applications built with Java Swing

## Technologies

* Java
* Java Swing
* Object-Oriented Programming (OOP)
* Cryptography Algorithms

This project demonstrates the implementation of various cryptographic techniques, including encryption, decryption, key generation, and simple cryptanalysis methods.
